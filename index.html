    Introduction

    This documentation is to walk through some of the features of our website and how to customize pages and modules

    For content look and feel, please check out our styleguide.

    Siteleaf CMS

    We use Siteleaf as our Content Management System (CMS). It is a third-party service that helps publish content onto our website. Contact Jayme for our login credentials.

    Responsive

    We implemented a responsive website so that we can control how our website displays over multiple devices.

    Breakpoint Target Examples
    320 Mobile Devices iPhone, Android
    600 Small Tablets Nexus 7, Amazon Fire
    768 Tablets iPad
    1024 Small Desktop Desktop

    Modules

    Banner Lead

    • Example of Usage: Homepage
    • Description: full width banner with video playing capability
    Meta Property Meta Value Description
    sectionTemplate banner-lead to activate this template
    sectionClass classes (space separated) add predefined classes to the template
    backgroundVideo video src add source url to show video (host video in rackspace)
    backgroundImage image src add source url to show image
    backgroundPosition XX% YY% position where the background image’s focal point is. A percentage value is used (horizontal% vertical%)

    Billboard

    • Example of Usage: Homepage
    • Description: a image/video slider
    • Subpage: Slides for the billboard are created by adding a subpages. There are two types of slides, backgroundImage or video type - both must be declared as a meta property.
    • background images should be in 16:9 format (ex. 1024x576)
    Meta Property Meta Value Description
    sectionTemplate banner-lead to activate this template
    sectionClass classes (space separated) add predefined classes to the template
    sectionStyles ex. background-color:red; add custom styles to the section wrapper
    video youtube or vimeo link must be in the following format:
    • Youtube: https://youtu.be/GBnxfTkICs
    • Vimeo: https://vimeo.com/128745697
    backgroundImage image src add source url to show image
    link href url add href url to make slide clickable. Only works when video is inactive.
    linkTarget ex. _blank open link in new tab

    Icon Over Text

    Note: this view is a subpage (child) of modules grid-2 or grid-3.
    • Example of Usage: Shop To Restore
    • Description: show either a icon (font awesome) or image over text.
    Meta Property Meta Value Description
    sectionTemplate icon-over-text to activate this template
    sectionClass classes (space separated) add predefined classes to the template
    link href url make the module clickable
    headerIconClass ex. fa fa-balance-scale add Font Awesome classes.
    Reference: font awesome
    headerImageUrl image src add source url to show image
    headerImageWidth XX% change the width of the image

    Image
